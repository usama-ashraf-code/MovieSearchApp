# 🎬 Movie Search App

Movie Search App is a responsive web application built with **HTML**, **CSS**, and **JavaScript** using the **TMDB (The Movie Database) API**. It allows users to browse popular movies, search for movies by title, and view detailed information such as movie posters, ratings, and release years. The application features a clean, modern, and responsive user interface that provides a smooth user experience across desktop and mobile devices.

## 🚀 Features

- Display Popular Movies
- Search Movies by Title
- Dynamic Movie Posters
- Movie Ratings
- Release Year
- Responsive Design
- Fast API Integration
- Clean & Modern UI
- Handles Missing Data Gracefully

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- TMDB API

## ⚙️ Installation

1. Clone the repository

```bash
git clone (https://github.com/usama-ashraf-code/MovieSearchApp.git)
```

2. Open the project folder.

3. Run `index.html` in your browser.

## 🌐 API

This project uses **The Movie Database (TMDB) API** to fetch real-time movie information.
